Packages

Salt as been packaged for a number of platforms, these packages can be downloaded via platform specific repositories.

  • RedHat Enterprise Linux / Centos (EPEL 5, EPEL 6)
  • Ubuntu (PPA)
  • Arch (AUR)
  • FreeBSD
  • Gentoo
  • Debian (upstream maintainer needed!)

PyPI

Salt is available on the Python Package Index and can be installed via pip or easy_install:
